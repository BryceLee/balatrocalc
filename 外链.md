# BalatroCalc 外链目标清单

更新时间：2026-05-26

原则：
- 不推广隐藏的 Calculator 3。
- 不做批量垃圾外链、评论区灌水、无关目录站。

- 每次投放都用一个具体场景承接，不只丢首页链接。

## 具体投放角度

### 1. Calculator 场景

标题：
- I made/maintain a Balatro score calculator workflow for checking boss blind breakpoints
- Quick way to test Joker order and hand score before risking a run

落地页：
- https://balatrocalc.com/

内容要点：
- 不是“请访问我的网站”，而是展示一个具体例子：当前手牌、Joker 顺序、盲注目标、计算结果。
- 说明适合检查 boss blind breakpoint、high ante hand、Joker order。
- 不提 Calculator 3。

### 2. Seed Analyzer 场景

标题：
- A practical workflow for reviewing Balatro seeds before replaying them
- Seed list vs seed analyzer: when each one helps

落地页：
- https://balatrocalc.com/balatro-seed-analyzer
- https://balatrocalc.com/blog/balatro-seed-analyzer-guide

内容要点：
- 把 seed list 和 route analysis 区分开。
- 举一个 seed 路线复盘例子：early shop、deck choice、boss blind、Joker priority。

### 3. High Card / Build 场景

标题：
- High Card build checklist for newer Balatro players
- How I decide whether a High Card run is worth committing to

落地页：
- https://balatrocalc.com/blog/balatro-high-card-build
- https://balatrocalc.com/balatro-builds

内容要点：
- 用教程页承接，不要直接把工具当广告。
- 可以顺带链接 calculator：最后一步才用于 score breakpoint。

### 4. Save / Mods 场景

标题：
- Balatro save location quick reference
- Clean way to install Balatro mods and keep save backups

落地页：
- https://balatrocalc.com/balatro-save-locations
- https://balatrocalc.com/how-to-install-balatro-mods

内容要点：
- 适合 Steam 讨论、中文社区、GitHub issue 回复。
- 链接必须出现在“解决具体问题”的上下文里。

## 建议执行顺序

1. GitHub awesome-balatro：准备 PR，最干净、最可持续。
2. Steam Guide：写一篇完整指南，放 2-3 个自然链接。
3. Reddit r/balatro：先发一个带具体例子的工具/教程帖，接受人工审核。
4. Balatro Wiki：走讨论/编辑规则，确认能否加外部工具。
5. Balatro HQ：提交工具收录。
6. 中文社区：用中文教程页做 2-3 篇内容分发。
