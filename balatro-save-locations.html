Save guide

Balatro save file location by platform

If you need to back up a run, replace profile.jkr, inspect meta.jkr, or open a save in the editor, the first job is finding the right folder. This page keeps the common Balatro save file locations for Windows, macOS, Linux, and Steam Deck in one place.

Open Save Editor

Common save paths

  • Windows: %APPDATA%/Balatro
  • macOS: /Users/[user]/Library/Application Support/Balatro/
  • Linux / Steam Deck: [steam path]/compatdata/2379780/pfx/drive_c/users/steamuser/AppData/Roaming/Balatro/

The folder can contain files such as profile.jkr, meta.jkr, settings.jkr, and numbered save files. If you are using cloud sync, close Balatro before replacing anything so Steam does not overwrite your backup.

Backup rule

Before you replace or edit any file, duplicate the entire Balatro save folder or at least copy the exact .jkr file you plan to touch. Backups are faster than recovery.

Quick checks by platform

1

Windows save folder

Press Win + R, paste %APPDATA%/Balatro, and copy the folder before opening any .jkr file in the save editor.

2

Steam Deck and Proton

Check the Steam library path that owns Balatro, then follow compatdata/2379780 into the Windows-style AppData folder. Editing a different prefix will not affect your live save.

3

macOS Library folder

If the Library folder is hidden, use Finder's Go to Folder command and paste ~/Library/Application Support/Balatro/.

FAQ

Which Balatro file is my profile?

Your profile data is usually stored in profile.jkr. Back it up before using a Balatro save file editor or moving unlock data between installs.

What is meta.jkr used for?

meta.jkr is one of the common Balatro data files players inspect when troubleshooting account-level or progression-related state. Treat it as sensitive save data and keep an untouched copy.

Why can I not find the Steam Deck folder?

Steam Deck paths depend on the library where Balatro is installed. Search that library for compatdata/2379780, then continue into pfx/drive_c/users/steamuser/AppData/Roaming/Balatro/.
