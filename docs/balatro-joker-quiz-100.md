# Balatro Joker Quiz - 100 Question Draft

Purpose: first-pass review set for a Balatro familiarity mini-game. This draft starts with Joker cards only, using Chinese questions and English card names so it can later be translated or mapped to the existing card catalog.

Format: `ID | type | difficulty | question | options | answer | explanation | review note`

Question types:
- `effect`: recognize a Joker's main effect.
- `synergy`: choose the most natural pairing or build direction.
- `order`: understand trigger timing, retriggers, copy rules, or scoring order.
- `scenario`: choose a likely good play in a stated context.

Daily mode recommendation: draw 5 questions with a seeded date algorithm, not pure random. Suggested daily mix: 1 effect, 2 synergy, 1 order, 1 scenario.

Feedback rule: do not include "all options are wrong" as an answer option. Put a separate "I think this is wrong" feedback box under each answered question and save the question ID, selected option, user explanation, and optional contact for admin review.
