What is a Seed in Balatro?

In Balatro, a seed is a short code (letters and/or numbers) that controls all of the random events in a single run. If two players use the same seed and the same starting deck, the order of shops, Jokers, Blinds, Boosters, and many other random elements will follow the same pattern.

You can think of a seed as the "DNA of a run": it does not change the rules of the game, but it decides how the random pieces are arranged.

What a Seed can do

  • Control the order of shops and which Jokers are eligible to appear.
  • Control the order of Blinds and Boss Blinds.
  • Control which Boosters and Planet cards show up and when.
  • Allow different players to replay or share the same style of run.

What a Seed cannot do

  • It cannot unlock Jokers, Planet cards, Vouchers, or Blinds for you.
  • It cannot force the game to use cards that you have not unlocked yet.
  • It does not change the basic rules, scoring, or difficulty settings.
Important: Balatro seeds only arrange the random events. If your save file does not have certain Jokers or cards unlocked, those items will not appear even if they are part of a popular seed. They will be replaced by other available options from your current unlock pool. For the most accurate reproduction of a seed, use a fully unlocked save file.

Seed & Unlocks – Quick FAQ

Why do I see different Jokers than someone else using the same seed?

Because seeds only control the order of random events. The actual pool of Jokers and cards you can see is limited by your unlock progress. If you have not unlocked a Joker, it cannot appear for you, even if the seed would normally use it.

Does a seed automatically unlock cards for me?

No. Seeds do not unlock anything. They only define how the game uses the content you have already unlocked in your save file.

How do I get a “perfect” reproduction of a seed?

To match another player’s run as closely as possible, you should use:

  • The same Balatro version (e.g. 1.0 / 1.1).
  • The same starting deck.
  • A save file with all Jokers, Planets, Boosters, and Blinds unlocked.

Without a full-unlock save, the seed will still work, but some Jokers or rewards may be different or replaced.
