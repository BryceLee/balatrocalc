Balatro 玩法指南（术语/机制）

Balatro 详细玩法 + 术语解释：Joker / Chips / Hand 一次看懂

Balatro 的核心循环很简单：用最少的出牌次数打出足够高的分，过关并滚雪球。难点在于你要理解：什么是“得分来源”，什么是“倍率来源”，以及 Joker 该如何围绕某个手牌方向构筑。

常用术语速查

  • Ante：关卡层级，越高要求越苛刻。
  • Blind / Boss Blind：每轮的得分目标与限制条件；Boss 会加额外规则。
  • Hand / Discard：每回合可出牌次数与可弃牌次数。
  • Shop：买 Joker、行星牌、塔罗牌、强化包、券（Vouchers）的地方。
  • Scaling（成长）：能随时间/局内行为变强的 Joker（越早拿越赚）。

Chips / Mult / XMult：三种“得分语言”

  • Chips：像“底分”。常见来自：手牌基础值、计分牌本身、增强/版本、部分 Joker。
  • Mult：像“加法倍率”。常见来自：手牌基础值、加 Mult 的 Joker、部分增强/版本。
  • XMult：像“乘法倍率”。它不是 +Mult，而是把当前 Mult 乘一下，通常决定上限。

想看更细的“结算顺序”，去读：Balatro 计分原理（Chips / Mult / Joker 顺序）。

Joker 类别（构筑思路）

你可以把 Joker 粗略分成几类，方便你在商店里快速做决策：

  • 得分型：直接给 Chips / Mult / XMult。适合补短板或当核心。
  • 成长型：会随着出牌、弃牌、回合、触发次数累积变强。越早拿越好，但也更吃路线规划。
  • 触发/复制型：让某些效果重复触发、复制其他 Joker、改变触发条件。强度与“目标 Joker”绑定。
  • 经济型：给钱、给折扣、给更多商店资源，间接提升强度（让你更快找到关键件）。
  • 工具型：改变牌库、改变牌型、提供容错（更多弃牌/出牌等），提升稳定性。

Hand 类型（常见牌型）

Balatro 的基础牌型来自扑克，并在后期扩展出更强的组合。常见的有：

  • High Card / Pair / Two Pair / Three of a Kind
  • Straight / Flush / Full House / Four of a Kind
  • Straight Flush
  • Five of a Kind / Flush House / Flush Five（更偏后期构筑）

每种 Hand 都有自己的“基础 Chips 与基础 Mult”，并且可以通过行星牌提升等级，让它变成你构筑的“主输出手”。

Chip 类型（得分来源）

除了 Hand 基础值之外，Chips 还经常来自：

  • 计分牌本身：被计分的牌会贡献 Chips。
  • 增强与版本：例如 Foil（+Chips）、Holographic（+Mult）、Polychrome（XMult）等。
  • 消耗牌与券：部分塔罗/行星、Vouchers 会改变牌库或提供额外加成。

新手最实用的三条建议

  1. 先把“主打手牌”定下来：你是走 Flush、Straight，还是 High Card 成长？越早确定越容易选到匹配的 Joker/强化。
  2. 优先补短板：Chips 不够就补 Chips，Mult 不够就补 Mult/XMult；别只堆一种。
  3. 注意顺序：尤其是有 XMult、复制、或依赖当前数值的 Joker，位置经常决定上限。
